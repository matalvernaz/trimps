Trimps Save Editor

Edit a Trimps save entirely in your browser — nothing is uploaded. In Trimps, open Settings and press Export to copy your save code. Paste it below, edit, then paste the new code back into Settings, Import.

Back up your original save code first. Editing to invalid values can make a save fail to load.

Paste a save code and press Decode Save.
