Trimps Tools

Accessible hosting of the community Trimps tools, tuned for screen readers. Each tool runs entirely in your browser.

  • Save editor — decode, browse, and edit any value in a Trimps save; linked values stay consistent automatically.
  • Perky — universe 1 perk optimizer by Grimy. Paste your save, pick a preset, get an allocation with a screen-reader results list.
  • Surky — universe 2 perk optimizer by nsheetz (UI adapted from justme). Results land in labeled fields with spoken run summaries.
  • zFarm — Grimy's zone-farming calculator (vendored as upstream; accessibility pass pending).

AutoTrimps automation is available inside the game's screen reader build via an opt-in button — it never loads unless you enable it.

  • Spire layout optimizer — pick floors, budget, and trap upgrades; get a best layout with a floor-by-floor spoken breakdown. Built on DataBeaver's trimps-tools.
